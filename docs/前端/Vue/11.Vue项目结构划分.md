# Vue项目结构划分

```markdown
src/
├── assets			
│   ├── css
│   └── img
├── common
├── components
│   ├── common
│   └── content
├── views
├── router
├── store
├── network
├── App.vue 
└── main.js
```

- assets：项目静态资源
  - css：项目公共css
  - img：项目公共图片
- common：项目公共js文件
- components：组件
  - common：公共组件（其他的项目也可以用）
  - content：项目公共组件
- views：界面视图组件
- router：路由
- store：VueX
- network：网络相关
- App.vue：Vue单实例对象
- main.js：项目主入口
