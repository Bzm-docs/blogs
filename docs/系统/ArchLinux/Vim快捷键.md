# Vim快捷键

[【Vim】使用map自定义快捷键](https://www.jianshu.com/p/8ae25a680ed7)

| 快捷键      | 作用                               |               |
| ----------- | ---------------------------------- | ------------- |
| Ctrl+r      | 取消撤销                           |               |
| Ctrl+f      | 下一页                             |               |
| Ctrl+b      | 上一页                             |               |
| Ctrl+w      | 分屏后移到                         |               |
| Ctrl+o（i） | 回到上一个光标所在位置             |               |
| x           | 删除当前选中的字符                 |               |
| c           | 删除字符并进入编辑模式             | cw，cb，ciw， |
| ci"         | 删除“”包裹的内容                   |               |
| w           | 光标移动到下一个词                 |               |
| b           | 光标移动到上一个词                 |               |
|             | 将这行保持为中心                   |               |
| f+字母      | 光标移到下面最近的这个字符         |               |
| :r !ls      | 将当前目录下所有文件名放到编辑器中 |               |
| :normal     | 将选中的所有行，执行后面的动作     |               |
|             |                                    |               |
|             |                                    |               |
|             |                                    |               |



查看这个键是否被映射

:verbose imap <tab>
