# Spring Boot 简介

​    `SpringBoot`是一种全新的框架，目的是为了简化`Spring`应用的初始搭建以及开发过程。该框架使用特定的方式(集成`starter`，约定优于配置)来进行配置，从而使开发人员不需要再定义样板化的配置。`SpringBoot`提供了**一种新的编程范式**，可以更加快速便捷地开发`Spring`项目，在开发过程当中可以专注于应用程序本身的功能开发，而无需在`Spring`配置上花太大的工夫。

>   简化`Spring`应用开发的一个框架；
>
>   整个`Spring`技术栈的一个大整合；
>
>   `J2EE`开发的一站式解决方案；
